


## 🌍 Global Fuel Pricing Dashboard (2020–Live)

![Status](https://img.shields.io/badge/status-active-success)
![Data](https://img.shields.io/badge/data-live%20%2B%20historical-blue)
![Tech](https://img.shields.io/badge/built%20with-Python%20%7C%20PowerBI%20%7C%20Streamlit-orange)

An end-to-end data analytics project that monitors and analyzes **global fuel prices from 2020 to present (live updates)**, combining historical datasets with real-time API integration.

---

##  Project Overview

This project delivers a dynamic analysis of global fuel prices (Petrol, Diesel, LPG) across multiple countries and regions.

 **Beyond tracking prices**, the project also:

* Analyzes **global economic conditions**
* Studies the **impact of world events** (e.g. inflation, supply shocks, geopolitical changes) on fuel prices
* Provides insights into how fuel prices evolve over time based on real-world factors

---

  [https://www.oilpriceapi.com/auth/signup-success](https://www.oilpriceapi.com/auth/signup-success)

---

##  Features

*  Interactive Power BI Dashboard
*  Global comparison by:

  * Country
  * Region
  * Income level
*  Time-series analysis (2020 → Present)
*  Live fuel price tracking (daily updates)
*  AI Chatbot:

  * Ask questions about the dataset
  * Get instant insights
*  Telegram Automation:

  * Daily fuel price summary notifications
*  Analysis of **global events impact on fuel pricing**

---

## Tech Stack

* **Python** → Data processing & API integration
* **Power BI** → Data visualization
* **Streamlit** → Interactive web app
* **REST API** → Live fuel data
* **Telegram Bot** → Automation
* **Kaggle** → Historical dataset
